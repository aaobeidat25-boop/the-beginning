# 🏧 ATM System Project - C++

### 🌟 نبذة عن المشروع
هذا أول مشروع برمجي لي باستخدام لغة *++C*. هو عبارة عن نظام صراف آلي (ATM) بسيط يحاكي العمليات الأساسية.

### 🛠️ الميزات (Features):
* *التحقق من كلمة المرور:* النظام لا يفتح إلا بكلمة السر الصحيحة.
* *الاستعلام عن الرصيد:* معرفة المبلغ المتبقي في الحساب.
* *السحب (Withdraw):* سحب مبلغ محدد مع التأكد من توفر الرصيد.
* *الإيداع (Deposit):* إضافة مبلغ جديد للرصيد.
